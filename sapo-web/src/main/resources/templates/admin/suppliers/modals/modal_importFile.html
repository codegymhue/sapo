Nhập dữ liệu nhà cung cấp

Chú ý:
- Mã nhà cung cấp phải là duy nhất đối với các nhà cung cấp độc lập
- Chuyển đổi file nhập dưới dạng .XLS trước tải dữ liệu.
- Tải file mẫu nhà cung cấp tại đây
- File nhập có dung lượng tối đa là 2MB và 5000 bản ghi.
Kéo thả file vào đây hoặc tải lên từ thiết bị
  • Tải File mẫu tại đây (cập nhật ngày: 01/01/2023)
  • File có dung lượng tối đa là 2MB và 5000 dòng
